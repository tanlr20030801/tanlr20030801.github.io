In the file optimized_result.mat, the variable 'Pattern' is a 1x100 cell in which each cell stores a 128x128 binary matrix representing a freedom shape (0 is dielectric and 1 is gold), the variable 'Period' is a 100x1 double (unit of nm) which corresponds to the period of the 100 metasurface units. In the file simulated_T.mat, the variable 'T_all' is a 100x2x301 matrix, which represents the simulated transmission curves of the 100 metasurface units using RCWA under the incident angle of 0 and 15 degree in the wavelength range of 450-750 nm (step of 1 nm).

For the electromagnetic simulation, the superstrate is quartz (n=1.52), the substrate is SOG (spin on glass, n=1.45). In the metasurface layer, the dielectric is SOG, the refractive index of gold is stored in Au_nk.txt. The thickness of metasurface layer is 20 nm (the thickness of superstrate and substrate is set to be infinite).
